Docker Configuration
Configure AOS locally with Docker (Ubuntu/Centos OS are supported) by running: git clone https://github.com/aosapp/account-service.git AOS installation: cd account-service/Nimbus_aos/aos_all_in_one/ chmod +x startup.sh ./startup.sh For the DevOps story: cd account-service/Nimbus_aos/aos_pipline/aos_all_in_one/ chmod +x startup.sh ./startup.sh * Review and update the relevant fields in .env file (hidden file).   Note: For machines in Micro Focus network, don't forget to update the proxy fields.   Requires Docker Compose: https://docs.docker.com/compose/install/#install-compose   Requires Docker App: https://github.com/docker/app#installation
